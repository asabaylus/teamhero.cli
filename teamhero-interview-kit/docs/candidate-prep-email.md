# Candidate Preparation Email Template

A template proctors can send to HR for forwarding to candidates **before**
the interview session. Replace the `{{...}}` placeholders before sending.

---

**Subject:** Preparing for Your Technical Interview – Setup Instructions

Hi [Candidate Name],

We're looking forward to your upcoming technical interview. To make the
most of our time together, please complete the setup below **before your
session**.

**What to expect**

You'll work on a small full-stack coding exercise in a live, proctored
session. The starter project is provided — you'll extend it to add
functionality. You may use your normal development tools, including
AI-assisted coding tools if that's part of your workflow.

The session is recorded (terminal + git history). You'll review and sign
a short privacy release before we begin.

**Prerequisites**

Please have the following installed and working:

- {{PREREQUISITES}} (filled by proctor based on stack)
- Git
- A code editor of your choice

**Setup steps**

1. Clone the repository: [Repository Link]({{REPO_URL}})
2. Follow the "Getting started" section in the README to install
   dependencies and verify the project runs
3. Confirm you can see the app running in your browser

If everything loads without errors, you're all set. **Do not begin
working on the exercise** — we'll walk through the task together at the
start of your session.

If you run into setup issues, reply to this email and we'll help you
troubleshoot before your session.

Looking forward to it,
[Your Name]
